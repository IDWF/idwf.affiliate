idwf.affiliate Installation
---------------------------

To install idwf.affiliate using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``idwf.affiliate`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        idwf.affiliate

* Re-run buildout, e.g. with:

    $ ./bin/buildout
